# EcoFoods Management App<br>
A virtual shop wich sells bio products.


## Database:<br>
Database was created in visual studio

### Products table:<br>	
- **id**Primary Key<br>
- **Name**Varchar<br> 	
- **Description**Varchar<br>
- **Price**Int<br> 
- **Quantity**Int

### User table:<br>
- **id**Primary Key<br>
- **Name**Varchar<br>	
- **Email**Varchar<br>	
- **Password**Varchar<br>	
- **Role**(custom implicit)

### Roles:<br>
- **Admin**<br>
- **Customer**
