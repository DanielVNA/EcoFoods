# EcoFoods Management App<br>
A virtual shop used to sell bio products.


## Database:<br>
The database was created in Visual Studio using migrations.

### Products table:<br>	
- **id**:Primary Key<br>
- **Name**:Varchar<br> 	
- **Description**:Varchar<br>
- **Price**:Int<br> 
- **Quantity**:Int

### Users table:<br>
- **id**:Primary Key<br>
- **Name**:Varchar<br>	
- **Email**:Varchar<br>	
- **Password**:Varchar<br>	
- **Role**

### Roles:<br>
- **Admin**<br>
- **Customer**
